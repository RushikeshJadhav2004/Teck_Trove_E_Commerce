##Setup Instructions

*Prerequisites*

Node.js
MongoDB installed locally or MongoDB Atlas

## Local Development
*Clone the repo:*
```
git clone https://github.com/your-username/techtrove.git
cd techtrove
```

*Setup server:*
```
cd server
npm install
npm run dev

```

*Configure .env in server/:*
```
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

```
